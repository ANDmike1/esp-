## 结构化优先级（快速模式默认）

未单独问卷时，采用以下默认取向（与「比赛迭代快 + 可焊性」一致）：

| 取向 | 本次默认 |
|------|----------|
| 低功耗 | 次要（有线供电为主） |
| 低成本 | **重要**（但不以牺牲母线裕量为代价） |
| 快速落地 / 可调试 | **最重要** |
| 国产供应链 | **倾向混合**：电源芯片可 TI/成熟料，功率 MOS/驱动按 LCSC 备货选 |

若与你不符，请回复要改的 1～2 条即可。

---

## [假设] 清单（需你事后确认）

1. 主控不在本板，本板仅 **电源 + 双路电机驱动 + 去耦/保护**。  
2. **5 V / 3.3 V** 峰值电流单路 **≤ 2 A** 量级足够（若带多舵机需上调）。  
3. 输入允许 **反接保护** 以保险丝 + TVS 为主；是否加理想二极管/ORing 未强制。  
4. **LM2596** 作为 TPS62135 的「易焊备选」仅针对降压部分；需重新画原理图与布局，非 pin-to-pin。
