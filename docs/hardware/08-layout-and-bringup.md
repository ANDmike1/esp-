---

## 1. 原理图收尾清单（DRC 前）

| 检查项 | 说明 |
|--------|------|
| **网络名统一** | **VM / 12V / GND / 5V / 3V3** 与 PCB 一致；避免同名不同网。 |
| **LM2596 固定输出** | **无 FB 分压电阻**（除非你用 ADJ）；**续流二极管方向**、**输入/输出电容极性**。 |
| **AT8236** | **VM**、**OUT**、**ISEN** 与 **2512 检流** 走线 **开尔文**；散热焊盘接 **GND**。 |
| **入口 SS54** | 若仍串联在总入口：与 **8 A 堵转** 是否匹配（见 `07-bom` §6）；否则标注 **「禁止反接」**。 |
| **无保险丝** | 原理图删除保险符号及多余网络，避免 DRC 悬空。 |

---

## 2. PCB 布局优先级（嘉立创 EDA）

| 优先级 | 区域 | 要点 |
|--------|------|------|
| P0 | **12 V VM / 电机电流** | **铺铜 + 过孔**；**不换层细瓶颈**；从 **端子 → 电解 → AT8236 VM** 最短。 |
| P0 | **LM2596** | **Vin 电容贴脚**；**肖特基紧贴 SW**；**电感靠近 SW**；**开关节点面积小**。 |
| P1 | **2512 检流** | **取样脚独立** 进出芯片 **ISEN**，不与电机大电流铜皮混一段再走分支。 |
| P1 | **3.3 V 至排针** | 按 **~1.2 A** 布宽或铺铜至 **ESP32 供电针**。 |
| P2 | **丝印** | **12V/GND/5V/3V3**、**电机方向**、**极性电容**。 |

---

## 3. 分阶段上电（强烈建议）

| 步骤 | 操作 | 判据 |
|------|------|------|
| S1 | **限流电源** 仅接 **12 V 入口**（**不接电机**） | 无冒烟；各轨 **对地短路电阻** 正常（非近似短路）。 |
| S2 | 测 **5 V、3.3 V 空载电压** | **5.0±裕量 / 3.3±裕量**；LM2596、二极管微热可接受。 |
| S3 | 接 **霍尔 5 V** | 编码器工作正常。 |
| S4 | 接 **ESP32 3.3 V**（可先低压限流） | 起机、屏、WiFi 粗测。 |
| S5 | 最后接 **电机 + 逐渐加大负载** | 堵转电流与温升可接受；检流波形（若有）合理。 |
