# 09 — 投产前核对清单（电机电源驱动板）

| 项目 | 嵌赛 — 电机电源驱动板 |
|------|------------------------|
| 日期 | 2026-05-06 |

**用途**：原理图/PCB 定稿后，**下单打样 + 备料焊接** 前逐项打勾，减少返工。

---

## A. 原理图（嘉立创 EDA）

- [ ] **D_IN**：已定为 **SK104B（SMB）**（或等同 **40 V / 10 A 档**），网络 **BAT+ → 二极管 → VM**。  
- [ ] **无保险丝**：库与 BOM 中已无悬空保险符号。  
- [ ] **LM2596×2**：固定 **5 V / 3.3 V**，续流二极管极性、**L5/L6**、**Cin/Cout** 极性与耐压已核对。  
- [ ] **AT8236×2**：**2512 检流**、**ISEN 开尔文**、**VM 退耦**齐全。  
- [ ] **运行 DRC**，飞线清零或与 intentional NC 一致。

---

## B. PCB

- [ ] **VM / 12 V**：铺铜与线宽满足 **~8 A**；过孔数量足够。  
- [ ] **SK104B**：本体下方及两极 **向外铺铜**（降低热阻）。  
- [ ] **LM2596**：开关节点紧凑；**TO-263** 散热焊盘与过孔。  
- [ ] **测试点**：**VM、5 V、3.3 V、GND**（至少各一处便于万用表）。  
- [ ] **丝印**：电源±、电机接口方向、极性电容。

---

## C. Gerber 与下单

- [ ] **导出 Gerber ZIP**（嘉立创规则已通过或未勾选致命项）。  
- [ ] **板厚 / 铜厚 / 阻焊颜色** 已选。  
- [ ] **SMT 贴片还是手焊**：TO-263 / SMB / 2512 / ESOP-8 工艺自检。

---

## D. 元件采购（对照 `07-bom.md`）

- [ ] **SK104B**（或你 BOM 中最终型号）数量 ≥1。  
- [ ] **LM2596 固定电压**、**0650 电感**、**2512 0R1 ≥2 W**、AT8236×2。  
- [ ] 电解 / 陶瓷耐压：**VM、Vin ≥25 V**；**5 V、3.3 V 输出电容** 匹配负载。

---

## E. 样板到货后（衔接 `08-layout-and-bringup.md`）

- [ ] 按 **§3 分阶段上电 S1→S5**，限流优先。
